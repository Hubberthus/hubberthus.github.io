Usage

This webpage was designed to be a quick and easy-to-use tool.

Of course if something is easy for me doesn't necessarily means that it's easy for others as well.

Home page overview

If you start up the home page, after the loading screen you can see three main sections:

- Navigation bar

On the top there are the main navigational and functional buttons and inputs.

For now, there is only one functional input: The module selector drop-down menu. This shows the current module, and of course by clicking on it the drop-down menu allows you to select from the list.

- Peripherals list

On the left, there is a list of available peripherals, defined by the MCU chip on the module.

By clicking on a peripheral, you can extend its details: a description panel, and a pin list.

The description panel is also extensible, because some of them can be long. You don't need to see it always, and is closed by default.

The pin list is only visible if the peripheral is turned on. For each pin you can see a:

  • Number, if the pin's position is fixed.
  • On/Off button, if the pin's position is fixed, but it is optional.
  • Drop-down menu, if you can select from different positions. "---" there means that it is also optional, and by selecting that it is turned off.

Peripherals can be turned on and off with the drop-down menu to the right of the their name. Sometimes there is no menu there, for example if the pins are not routed out, or the peripheral is already included on the board.

If multiple modes can be selected for a peripheral, then instead of an "ON" option, there are a list of modes from which you can choose.

- Module view

Most of the screen is filled with the WYSIWYG view of the module pin settings. It consists of an image of the module, and the list of pins at its sides, each pin showing the assigned function to it.

With the help of the image, you can see exactly where to attach the wires of given peripherals.

The round labels next to the image are the pin names. These are also buttons. Click on any of them, and over the image you will see some detailed information about that particular pin. What is the official description for it, is there anything attached to it on the board (LED, resistor, capacitor...), and if it is directly routed out from the MCU, there are details if it is input/output/PWM capable, etc.
